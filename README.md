# PRODIGY_WD_02
STOPWATCH WEB APPLICATION
A clean, responsive Stopwatch built using HTML, CSS, and JavaScript.

## Features
- **Start/Pause**: Accurately track time using `setInterval`.
- **Reset**: Clear the timer and lap history.
- **Lap Tracking**: Record specific time intervals during a session.

## Technologies Used
- HTML5
- CSS3 (Flexbox for layout)
- Vanilla JavaScript (Date.now() for precision)
